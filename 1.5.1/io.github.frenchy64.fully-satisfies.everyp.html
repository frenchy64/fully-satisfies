everyp
(everyp)(everyp p)(everyp p1 p2)(everyp p1 p2 p3)(everyp p1 p2 p3 & ps)
Combines predicates into a variable-arity conjunction.

Operationally equivalent to:

  (defn everyp [& ps]
    (fn [& args] (every? #(every? % args) ps)))
